💡 Project Overview:
Developed an interactive application designed to help users learn new languages through structured lessons, practice, and quizzes.

🧠 Key Features:
• Learn words, phrases, and sentences in a selected language
• Daily lessons and flashcards with translations and pronunciations
• Interactive quizzes and practice tests to track progress
• Organized sections like vocabulary and grammar
• Clean, intuitive, and user-friendly interface
• Data storage using local storage / backend integration (Firebase concept)
